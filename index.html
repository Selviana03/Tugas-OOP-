1. Perbedaan Utama Antara Kelas dan Objek

  • Kelas (Class): Adalah "blueprint" atau cetakan dasar yang mendefinisikan atribut (data) dan perilaku (metode) dari suatu entitas. Ia belum memiliki data nyata (masih berupa rancangan).
  • Objek (Object): Adalah bentuk nyata atau hasil instansiasi dari Kelas. Objek sudah memiliki data yang spesifik.

2. 3 Keuntungan Menggunakan OOP

  1. Modularity (Modul): Kode dibagi menjadi unit-unit (objek) yang lebih kecil sehingga lebih mudah dikelola.
  2. Easier Maintenance (Pemeliharaan Mudah): Jika ada kesalahan (bug), lebih mudah dicari karena struktur kodenya terorganisir per kelas.
  3. Reusability (Penggunaan Kembali): Kode yang sudah dibuat bisa digunakan kembali di bagian lain atau proyek lain.

3. Implementasi Kode Dart (Class Hewan)

Berikut adalah kode Dart untuk mendefinisikan kelas Hewan dengan atribut nama dan metode bersuara():

// Definisi Kelas Hewan dalam Dart
class Hewan {
  // Atribut
  String nama;

  // Konstruktor
  Hewan(this.nama);

  // Metode
  void bersuara() {
    print("$nama bersuara: Meow... Guk... Rauurrr!");
  }
}

void main() {
  // Membuat objek baru (instansiasi)
  var hewanku = Hewan("Kucing");

  // Menampilkan atribut
  print("Nama Hewan: ${hewanku.nama}");

  // Menjalankan metode
  hewanku.bersuara();
}

Jawaban Soal No. 3:

Membuat Class Hewan dengan atribut nama dan metode bersuara().

// Definisi Class Hewan dalam Dart
class Hewan {
  // Atribut
  String nama;

  // Konstruktor
  Hewan(this.nama);

  // Metode
  void bersuara() {
    print("$nama sedang bersuara: Rauuurrr!");
  }
}

void main() {
  // Membuat Objek dari Class Hewan
  var kucing = Hewan("Kucing");
  
  // Mengakses Atribut dan Metode
  print("Nama Hewan: ${kucing.nama}");
  kucing.bersuara();
}
